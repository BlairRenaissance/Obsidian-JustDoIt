
| 初始化方式   | 语法形式         | 特点       | 适用场景          |
| ------- | ------------ | -------- | ------------- |
| 默认初始化   | `T a;`       | 基本类型不初始化 | 需要延迟初始化时      |
| 拷贝初始化   | `T a = b;`   | 可能产生临时对象 | 简单类型初始化       |
| 直接初始化   | `T a(b);`    | 直接调用构造函数 | 显式指定构造参数时     |
| 直接列表初始化 | `T a{b};`    | 禁止窄化转换   | C++11 推荐初始化方式 |
| 拷贝列表初始化 | `T a = {b};` |          |               |
| 值初始化    | `T a{};`     | 保证零初始化   | 需要确定初始值时      |

## 传统初始化方式

1. **int a;**
	默认初始化(default-initialization)
	- 未提供初始值
	- 对于基本类型(int)，不会自动初始化为0（值不确定）
	- 对于类类型，会调用默认构造函数

2. **int b = 5;**
	拷贝初始化(copy-initialization)
	- 使用等号(=)进行初始化
	- 先创建临时对象，再通过拷贝构造初始化目标对象
	- 对于基本类型(int)，与直接初始化效果相同

3. **int c(6);**
	直接初始化(direct-initialization)
	- 使用圆括号()进行初始化
	- 直接==调用构造函数初始化==对象
	- 效率通常比拷贝初始化高

## 现代初始化方式（推荐）

4. **int d{7};**
	直接列表初始化(direct-list-initialization)
	- 使用花括号{}进行初始化
	- C++11引入的现代初始化语法
	- ==禁止窄化转换==，更安全

5. **int e{};**
	值初始化(value-initialization) / 零初始化(zero-initialization)
	- 使用空花括号{}初始化
	- 对于基本类型(int)，会被初始化为0
	- 对于类类型，会调用默认构造函数

``` cpp
// 类初始化示例
class MyClass {
    int val = 10;  // 非零默认值
};
MyClass obj{};  // 值初始化使用默认值10
```
