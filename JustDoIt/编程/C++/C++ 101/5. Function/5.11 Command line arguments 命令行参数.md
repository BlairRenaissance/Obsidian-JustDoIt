## 使用命令行参数

```cpp
int main(int argc, char* argv[])
```

有时也会写成：

```cpp
int main(int argc, char**argv)
```

`argc`是一个整数参数，包含传递给程序的参数数量（argc = argument count）。`argc`将始终至少为 1，因为第一个参数始终是程序本身的名称。用户提供的每个命令行参数都会使`argc`增加 1。

`argv`是存储实际参数值的地方（argv = argument values，尽管正确的名称是 “argument vectors”）。尽管`argv`的声明看起来很吓人，但`argv`实际上只是一个 C 风格的字符指针数组（每个指针都指向一个 C 风格的字符串）。这个数组的长度是`argc`。

注意，我们不能使用基于范围的 for 循环来迭代`argv`，因为基于范围的 for 循环不适用于衰减的 C 风格数组。

## 解析命令行参数

当你在命令行上输入内容时，操作系统负责解析该请求。通常，操作系统对于如何处理双引号和反斜杠等特殊字符有特殊规则。

```plaintext
MyArgs Hello world!
```

```plaintext
共有 3 个参数：
0 C:\MyArgs
1 Hello
2 world!
```

通常，用双引号括起来的字符串被视为同一字符串的一部分：

```plaintext
MyArgs "Hello world!"
```

```plaintext
共有 2 个参数：
0 C:\MyArgs
1 Hello world!
```

大多数操作系统允许你通过在双引号前加反斜杠来包含文字双引号：

```plaintext
MyArgs \"Hello world!\"
```

```plaintext
共有 3 个参数：
0 C:\MyArgs
1 "Hello
2 world!"
```
